!ConEditor

こねでぃた(ConEditor)はフォルダに保存された複数のテキストファイルを一度に変更することができるテキストエディタです。

現在は開発途中版につき、意図しない不具合によって編集中のデータ破壊等が発生する可能性がありますので、編集するファイルのバックアップには十分にお気を付けください。


*アウトライン
「Outline」タブのチェックボックスでアウトライン機能を有効にできます。
アウトラインは行頭にが"*"(の連続)で始まる個所を抽出します。



*Git連携
GitForWindows(https://git-for-windows.github.io/)をインストールしている場合、
Git連携機能を使用できます。
Git連携を使用することによってエディタからコミットを行うことができます。


*ライセンス
好きに使ってください。
zlibライセンスとします。
